# nba-info

Some programs that examine elements of websites related to daily fantasy basketball and scrapes information from those sites. These sites include:
1. [Hashtag Basketball Defense vs. Position](https://hashtagbasketball.com/nba-defense-vs-position)
2. [RotoWire NBA Lineup Optimizer](https://www.rotowire.com/daily/nba/optimizer.php?site=DraftKings)
3. [RotoWire NBA Daily Lineups](https://www.rotowire.com/basketball/nba-lineups.php)

## Software Required
1. [Python](https://www.python.org/downloads/)
2. [Selenium](https://selenium-python.readthedocs.io/installation.html)
